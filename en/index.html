Open-PLC is a modular, distributed open source management system. Module is the basic unit of creating systems.
The control system, or the automation system, are modules integrated into a single network in various ways. Each module performs certain tasks.
The system is created from modules interacting as a master-slave. The master module manages the work of subordinates.
Master modules can be combined into a distributed, non-hierarchical network.
To start creating your own system, minimal knowledge and skills are enough.
For example, to implement the ideas of "smart home" you can start without a project, but create a system as the ideas and experience begin, starting with the simplest module.
 
The system can be used to solve various tasks, for example, "smart house", IoT, industrial automation, ...

The system has an open architecture that makes it easy to integrate with various solutions, both hardware and software.

The system is scaled from a simple standalone running module, to complex network solutions. The beginning of your own project can be a discovery board - "Blue Tablet"
A system module is a collection of hardware and software components.
The master module can be anything from a simple microcontroller to a computer with an operating system:
• based on STM32Fxxx without operating system
• Raspberry PI and the like with Linux
• x86 with different operating systems
When creating subordinate modules we will use STM microcontrollers (STM32F10x, STM32F30x). The simplest evaluation board is the "Blue Tablet" STM32F103C8T6.

As an example, various modules for Arduino can be used as sensors / actuators.
 
The description of programming will be replenished on the site as the software is created.

The script of modules operation can be written in "C", "C ++", programming languages ​​IEC 61131.

We a create firmwares for different purposes for slave modules. Firmwares of modules are unified, the user interactively configures them for their requirements.

All that is described on the site are components of a single system. But each component or idea can be used both autonomously and as part of its own project.

If the proposed hardware or software functionality is not enough, then we can develop and produce individual versions of your modules for various purposes, from solutions for home, to industrial.
